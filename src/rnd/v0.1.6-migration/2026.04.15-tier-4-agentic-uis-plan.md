## Context

Backend exposes a family of long-running agentic jobs (deep research, podcast generator, presentation generator, SWE team, bug-fix expediter, test suite, chained jobs) + an IO file server for fetching produced artifacts (mp3s, slides, reports). Mobile has no concept of any of these.

## Scope (to expand)

### Agentic Job Submission (8 endpoints)
- `POST /api/deep-research/submit` + health + report
- `POST /api/podcast-generator/submit`
- `POST /api/presentation-generator/submit`
- `POST /api/deep-research-to-podcast/submit` (chained)
- `POST /api/deep-research-to-presentation/submit` (chained)
- `POST /api/swe-team/submit`
- `POST /api/bug-fix-expediter/submit`
- `POST /api/test-suite/submit`
- `POST /api/test-fix-expediter/resume-from`

### Artifact Serving (2 endpoints)
- `GET /api/io/file` — fetch generated podcast mp3, presentation pptx/pdf, research report md
- `GET /api/io/health`

### Stats (2 endpoints)
- `GET /api/stats/time-saved` (per-user)
- `GET /api/stats/time-saved/global`

## UI Surface
- Per-job-type submission forms (with required runtime args)
- Unified "agentic jobs" dashboard reusing Tier-3 queue UI
- Artifact viewers (audio player for podcasts, slide viewer for presentations, markdown reader for research)
- Time-saved stats dashboard
