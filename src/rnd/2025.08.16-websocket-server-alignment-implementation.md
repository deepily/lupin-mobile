# WebSocket Server Alignment Implementation

**Date**: 2025.08.16  
**Project**: Lupin Mobile WebSocket Architecture Update  
**Status**: In Progress  
**Priority**: Critical  

## Executive Summary

The Lupin Mobile app's WebSocket implementation requires major updates to align with the Lupin FastAPI server's dual-session WebSocket architecture. Current implementation uses incorrect endpoints, authentication protocols, and session formats that prevent proper communication with the server.

## Critical Issues Identified

### 1. **ENDPOINT MISMATCH - BLOCKING ISSUE**
- **Current**: Mobile connects to `/ws/{session_id}` 
- **Problem**: This endpoint DOES NOT EXIST on the server
- **Required**: 
  - Main connection: `/ws/queue/{session_id}`
  - Audio connection: `/ws/audio/{session_id}`

### 2. **AUTHENTICATION TYPE MISMATCH - BLOCKING ISSUE**
- **Current**: Mobile sends `type: 'auth'`
- **Problem**: Server expects `type: 'auth_request'`
- **Required**: Update message type and add subscription array

### 3. **SESSION FORMAT MISMATCH**
- **Current**: May use underscore format ("wise_penguin")
- **Required**: Space format ("wise penguin")
- **Impact**: Session validation may fail

### 4. **EVENT SYSTEM DISCONNECT**
- **Current**: Limited TTS-focused event handling
- **Missing**: Queue events, notification events, system events
- **Required**: Comprehensive event handling for all server event types

## Technical Specifications

### Authentication Message Format
```dart
// CURRENT (INCORRECT):
{
  'type': 'auth',
  'token': 'mock_token_email_$userId',
  'session_id': sessionId,
}

// REQUIRED (CORRECT):
{
  'type': 'auth_request',
  'token': 'Bearer mock_token_email_$userId',
  'session_id': sessionId,  // Must be "adjective noun" with space
  'subscribed_events': [],  // Empty = receive all events
}
```

### WebSocket Connection URLs
```dart
// Queue Connection (Main UI)
final queueUri = Uri.parse('${AppConstants.wsBaseUrl}/ws/queue/$sessionId');

// Audio Connection (TTS Streaming)
final audioUri = Uri.parse('${AppConstants.wsBaseUrl}/ws/audio/$sessionId');
```

### Session ID Format
```dart
// Valid Examples:
"wise penguin"
"clever dolphin"
"happy turtle"

// Invalid Examples:
"wise_penguin"  // Underscore not allowed
"wise"          // Missing second word
"wise penguin cat"  // Too many words
```

### Event Types to Handle
```dart
// Queue Events
queue_todo_update, queue_running_update, queue_done_update, queue_dead_update

// TTS/Audio Events  
tts_job_request, audio_streaming_chunk, audio_streaming_status, audio_streaming_complete

// Notification Events
notification_queue_update, notification_play_sound

// System Events
sys_time_update, sys_ping, sys_pong

// Authentication Events
auth_request, auth_success, auth_error, connect

// Control Events
update_subscriptions, subscription_update
```
